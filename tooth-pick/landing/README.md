# ToothPick Landing Page

Página de aterrizaje moderna y responsiva para la plataforma dental ToothPick.

## 🚀 Características

- ✅ Diseño moderno y responsivo
- ✅ Optimizado para SEO
- ✅ Animaciones suaves y atractivas  
- ✅ Compatible con todos los navegadores
- ✅ Tiempo de carga ultra rápido
- ✅ Formularios de contacto integrados
- ✅ Analytics y métricas

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con gradientes y animaciones
- **JavaScript** - Interactividad y animaciones
- **Google Fonts** - Tipografía Inter
- **Responsive Design** - Mobile-first approach

## 📱 Compatibilidad

- ✅ Chrome/Edge/Firefox (últimas versiones)
- ✅ Safari (iOS/macOS)
- ✅ Dispositivos móviles y tablets
- ✅ Accesibilidad web (WCAG 2.1)

## 🏃‍♂️ Desarrollo

### Instalación

```bash
cd landing
npm install
```

### Servidor de desarrollo

```bash
npm run dev
```

La página estará disponible en `http://localhost:3001`

### Producción

```bash
npm run start
```

## 📊 Métricas de Performance

- **First Contentful Paint**: < 1.5s
- **Largest Contentful Paint**: < 2.5s
- **Time to Interactive**: < 3.0s
- **Cumulative Layout Shift**: < 0.1

## 🎨 Diseño

### Colores principales
- Primario: `#667eea` (Azul)
- Secundario: `#764ba2` (Púrpura)
- Fondo: `#f8fafc` (Gris claro)
- Texto: `#1a202c` (Gris oscuro)

### Tipografía
- **Inter**: Fuente principal
- Pesos: 300, 400, 500, 600, 700, 800
